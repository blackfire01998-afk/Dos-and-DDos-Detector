# Dos-and-DDos-Detector

Real-Time Network Traffic Monitoring & Attack Prevention

**Install dependencies**
    
    sudo apt update
  
    sudo apt install python3-pip iptables libpcap-dev tcpdump -y
  
    pip3 install scapy flask


  
   -Live traffic monitoring system
       
   -Automatic attack detection engine
        
   -Firewall based IP blocking mechanism
        
   -Interactive web visualization dashboard


**Automatic Response Process**

 * Identify attacker IP

 * Verify against whitelist
  
 * Block using iptables
  
 * Set auto-unblock timer
  
 * Log attack details
